How it works: The tool will divide your data into time intervals and average all data within each interval. For example, with 60-second intervals, it averages all data from 0-60 seconds, then 60-120 seconds, etc.

Data Processing Info
Processing: All data within each time interval will be averaged together. The Elapsed Time column will be set to the interval boundary (e.g., 60, 120, 180 seconds).
Output Preview
Result: You'll get one averaged row per time interval, with all numeric columns averaged and Sample No/Time from the last sample in each interval.
