About Location Configuration
Weather Data Source

Locations are used to fetch current weather data from the Open-Meteo API. This data is used to determine seasons and calculate appropriate slopes for diagnostic codes.

Default Location

The default location is used for weather-based slope calculations when enabling diagnostic codes. Only one location can be set as default at a time.
